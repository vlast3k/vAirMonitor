What is UxFleet?

UxFleet provides a broad range of unmanned systems planning capabilities. We realize that one-size never fits all. Rather than trying to provide a “throw-it-over-the-fence” planner capability, we have developed a set of tools that allow for pre-planning, runtime-planning, and re-planning.

Plan
Planning activities is critical. This means being able to set up a series of tasks that your unmanned systems must complete. The UxFleet task editor allows you to lay out a plan visually in a platform-agnostic way to provide reusability and portability.
Collaborate
At the heart of UxFleet’s collaboration technology is our Mission Management Service. This is a small, lightweight software server that runs either directly on your robot or on a remote server, responsible for orchstrating behaviors on each robot, and between robots in reaction to external and internal events.
Customize
A key advantage of UxFleet is that we are platform agnostic. If you have a robot (air, ground, surface, or underwater), we can adapt UxFleet to work with it. UxFleet can work with platforms with a broad range of autonomy, from systems that have only assisted teleoperation, to the latest UAVs with advanced waypoint and sensor integration capabilities.
Interact
UxFleet provides many different ways for you to interact with your unmanned systems. Whether you prefer to use the UxFleet desktop, tablet, or web-based user interface, you can be in control of your unmanned system in any type of work environment.
Analyze
At the heart of the UxFleet data analytics capabilities is our UxBuilder platform. In addition to incorporating open source solutions such as OpenCV and Point Cloud Library (PCL), UxBuilder includes Neya’s own implementations of state of the art machine learning algorithms.
Process
Depending on the number of robots you have, the quantity of data, and whether or not you need real-time processing, a cloud-based solution may be more appropriate. UxFleet, in combination with our UxBuilder product, supports automatic import of UGV and UAV data to Amazon cloud EC2 instances, where our UxBuilder algorithms can process your data.

White Paper: UxFleet Overview

UxFleet is an all-in-one mission planning and management tool for multiple unmanned assets. The UxFleet tool suite provides a user with the capabilities to intuitively design, simulate and monitor a complex mission scenario made up of many different tasks and involving both manned and unmanned units.

White Paper: UxFleet Task Editor Overview

UxFleet’s Task Editor is part of a suite of tools developed to help build mission plans that can be executed by manned or unmanned systems collaboratively. The Task Editor allows a mission designer to specify the process flow and the data passing between different manned or unmanned services to execute a particular task.
